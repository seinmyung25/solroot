## SolRoot Frontend

### Tech Stack

   1. react ^18.2.0
   2. vite $4.4.5
   3. web3 ^4.1.2
   4. cypress ^13.3.1 for testing
   5. typescript ^5.0.2
   6. tailwindcss ^3.3.3 for styling

### Installation

```shell
   https://github.com/curg/SolRoot.git
   cd client 
   # if client is not exist 
   git checkout solroot-frontend
   yarn # install dependencies
```

### Execution

```shell
   yarn dev
```

### Build

> we use swc

```shell
   yarn build
```
